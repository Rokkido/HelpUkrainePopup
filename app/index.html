Help Ukraine Popup

A small plugin that will help collect donations to help Ukraine.


Download

  • • Production version
  • • GitHub

Install (cdn)

<script src="https://cdn.jsdelivr.net/gh/Rokkido/HelpUkrainePopup/dist/hup.min.js"></script>

Start using

<script>hup();</script>
